所以可以看出，Generator 函数的特点就是

  1. 分段执行，可以暂停
  2. 可以控制阶段和每个阶段的返回值
  3. 可以知道是否执行到结尾
      function* gen() {
        yield 1
        yield 2
        yield 3
        return '雪月'
      }

      const g = gen()
      console.log(g.next())
      console.log(g.next())
      console.log(g.next())
      console.log(g.next())
