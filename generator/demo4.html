for of 循环

文章链接

for...of循环可以自动遍历Generator函数时生成的Iterator对象，且此时不再需要调用next方法。

      function* foo() {
        yield 1
        yield 2
        yield 3
        yield 4
        yield 5
        return 6
      }

      let a = foo()

      for (let v of a) {
        console.log(v)
      }
      // 1 2 3 4 5

利用Generator函数和for...of循环，实现斐波那契数列的例子。

      function* fibonacci() {
        let [prev, curr] = [0, 1]
        for (;;) {
          [prev, curr] = [curr, prev + curr]
          yield curr
        }
      }

      for (let n of fibonacci()) {
        if (n > 1000) break
        console.log(n)
      }
